The *professional plugins* are added to the existing *base plugins* and configured for [archetypes](/#archetypes) of
type **Advanced** and **Full**:

- [org.apache.maven.plugins:maven-invoker-plugin][0]
- [org.jacoco:jacoco-maven-plugin][1]
- [org.pitest:pitest-maven.version][2]
- [org.pitest:pitest-junit5-plugin][3]

[0]: https://maven.apache.org/plugins/maven-invoker-plugin
[1]: https://www.eclemma.org/jacoco/trunk/doc/maven.html
[2]: https://pitest.org/quickstart/maven/
[3]: https://github.com/pitest/pitest-junit5-plugin
